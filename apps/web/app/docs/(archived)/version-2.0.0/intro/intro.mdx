By combining the power of **Next.js**, **TailwindCss** and **Shadcn/ui**, you can build modern and responsive documentaion pages.

Criar documentações de forma rápida e fácil com organização automática e sem preocupações.

## Features

- Support for `.mdx` files with React components
- Styling powered by TailwindCss
- Automatic sidebar with organized categories
